## 配置环境变量
### JAVA_HOME
```
#输入你的JDK路径，下面是做演示
C:\Program Files\Java\jdk1.8.0_201
```
### CLASSPATH

```
.;%JAVA_HOME%\lib
```
### MAVEN_HOME
```
#输入你的MAVEN路径，下面是做演示
C:\Users\l1091\Documents\i-tool\apache-maven-3.6.0
```
### PATH
```
;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;%MAVEN_HOME%\bin
```
